Destiny Duel

Role : Independent Developer
Platforms : Steam, Windows
Engine : Unity
Language : C#
Development Time : 8 Months (August 2022 - March 2023)
Genres : Multiplayer, PvP, Strategy, Psychological, Turn-Based

Game Info

Destiny Duel is a PvP Psychological Strategy game. The gameplay is similar to Rock Paper Scissors in that you are trying to predict your opponent while being unpredictable yourself, but Destiny Duel has a lot more depth due to the many different possible situations.
The game features only 4 controls/moves that cost stamina to use: Attack, Guard, Recover, and Dodge. The goal of the game is to use these moves to reduce your opponent's health to 0.
Because there are only 4 controls, the game is easy to pick up and play with friends. For more competitive players, there is also a rank system with 10 ranks and 3 subranks.

My Experience

After developing Destiny Duel, I learned how to integrate Steam features in a game. This includes functions like saving data on the Steam cloud using Steam stats, adding Steam friends from within the game, and unlocking Steam achievements. I also learned how to make (more) appealing UI, which includes smooth animations and transitions.
During development, some challenges I overcame include getting the rank system and Steam features to work properly.
