Genres : Multiplayer, PvP, Sports, FPS

Game Info

Blox Ball Battlefield is an online PVP Sports game with gameplay based on dodgeball.
In this game, players have unlimited ammo, a dash, a once per game shield, and a once per game ghost ability.
Players can battle casually in a 12 player FFA, or duel competitively in teams of up to 3 - which grant (ELO) rating to the winners.
Whether you want to climb up the global leaderboard, or play casually with friends: Blox Ball Battlefield is the premier dodgeball game on Roblox.

My Experience

After developing Blox Ball, I learned how to implement support for mobile devices, server-client networking, and flexible GUI.
Some challenges I overcame include getting the game to be playable on mobile devices and handling players leaving/joining the server. There were collision issues and bugs when resetting the core game loop as well - which I was able to fix.
